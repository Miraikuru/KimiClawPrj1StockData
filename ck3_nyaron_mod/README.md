# Nyaron: The Feline Kin - Crusader Kings III Mod
# 喵灵：来自星辰的猫科类人生物

## 背景故事

在遥远的未来，闪耀科技世界（Stellar Tech World）的科学家们创造了一种全新的基因改造生物——**喵灵（Nyaron）**。他们是猫科动物与人类的完美结合，拥有：

- **出众的外貌** - 天生具有吸引力
- **超凡的战斗力** - 力量、速度和战斗本能远超常人
- **优秀的体力** - 适合各种体力劳动
- **讨好人类的本能** - 善于社交和取悦他人

但代价是：
- **学习能力较弱** - 难以掌握复杂的知识
- **管理能力不足** - 不擅长行政和治理

由于一次意外事故，一批喵灵被投放到了古典世界星球——也就是我们所知的地球。他们必须在这个原始的世界中生存，同时保持对自己起源的记忆。

## Mod特色

### 1. 遗传特性系统
- **喵灵（Nyaron）** - 基础遗传特性，可遗传给后代
- **始祖喵灵（Nyaron Progenitor）** - 仅第一个喵灵拥有
- **纯血喵灵（Nyaron Pureblood）** - 父母都是喵灵时获得
- **喵灵战士/侍从** - 专精分支特性

### 2. 独特文化
- **星际起源** - 基因记忆中保留了先进科技的知识
- **猫科敏捷** - 战斗中的速度和机动性优势
- **基因记忆** - 通过血脉传承知识

### 3. 宗教信仰
- **星际追忆** - 对起源星球的怀念
- **猫科神圣** - 认为喵灵形态是神圣的
- **适应性** - 在这个原始世界生存的智慧

### 4. 特色兵种
- **喵灵战士** - 精英战斗单位
- **喵灵猎手** - 快速打击单位
- **喵灵守护者** - 防御专精单位

### 5. 故事事件

#### 通用喵灵事件（所有喵灵角色）
- 喵灵觉醒 - 意识到自己的特殊身份
- 星之回忆 - 梦到起源星球
- 狩猎本能 - 猫科狩猎天性的觉醒
- 社交困境 - 与人类相处的挑战

#### 始祖专属事件（仅第一个喵灵）
- 降临新世界 - 初始事件
- 建立据点 - 创建喵灵 sanctuary
- 基因传承 - 教导后代
- 寻找同类 - 探索其他喵灵

## 游戏玩法

### 作为喵灵玩家
1. 利用你的**高战斗力和魅力**在战场上和宫廷中取得优势
2. 通过**生育**快速扩展家族（生育周期比人类短）
3. 建立**喵灵专属文化**和**宗教**
4. 训练**特色兵种**征服敌人

### 作为始祖喵灵
1. 你是所有喵灵的源头，你的后代将继承你的基因
2. 建立最初的喵灵据点
3. 决定喵灵与人类的互动方式（共存或征服）
4. 寻找其他可能幸存的喵灵

## 安装方法

1. 下载mod文件
2. 解压到 `Documents/Paradox Interactive/Crusader Kings III/mod/`
3. 启动游戏，在启动器中启用mod
4. 开始游戏，选择喵灵角色或等待喵灵角色出现

## 推荐玩法

1. **历史模式** - 选择867或1066年开局，喵灵作为稀有遗传特性随机出现
2. **喵灵崛起** - 选择始祖喵灵，从零开始建立喵灵帝国
3. **观察模式** - 观看AI喵灵家族如何在人类世界中生存发展
