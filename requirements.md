# 每日财经新闻简报需求文档

**版本**: v1.0  
**创建日期**: 2026年2月17日  
**最后更新**: 2026年2月17日

---

## 📋 需求概述

每日自动生成一份综合新闻简报，包含汇率、财经新闻、社交媒体趋势、游戏资讯等内容，以HTML格式呈现并推送到飞书。

---

## 一、汇率与贵金属

### 1.1 汇率数据
| 货币对 | 说明 |
|--------|------|
| 美元兑人民币 (USD/CNY) | 必须包含 |
| 日元兑人民币 (JPY/CNY) | 必须包含 |

**❌ 不需要**: 欧元兑人民币

### 1.2 贵金属价格
| 指标 | 说明 |
|------|------|
| 黄金(元/克) | 每克黄金兑人民币价格 |
| 伦敦金价 | 美元/盎司 |

**❌ 不需要**: 印度金价、白银（可选）

### 1.3 数据源要求
- **汇率**: 中国银行外汇牌价 API 或第三方汇率 API（exchangerate-api、fixer.io）
- **金价**: 上海黄金交易所（人民币/克）、伦敦金银市场协会（LBMA）
- **更新频率**: 每日上午9点前获取最新数据

---

## 二、财经新闻（6大领域）

每个领域至少2-3条新闻，必须事实核查确保时效性。

### 2.1 AI人工智能
- OpenAI、GPT、大模型等
- 不局限于特定公司

### 2.2 软件互联网
- 谷歌、微软等行业动态
- 全球软件行业趋势

### 2.3 硬件半导体
- 芯片行业动态
- 三星、SK海力士等

### 2.4 航天太空
- SpaceX、星链
- 中国航天进展

### 2.5 贵金属大宗商品
- 黄金、白银市场
- 有色金属

### 2.6 国内外时事
- 美股动态
- 中国经济政策

### 2.7 数据源要求
- **禁止**: 简单搜索后直接写入
- **必须**: 使用新闻媒体API或爬虫抓取
- **必须**: 对所有新闻进行事实核查，排除过时新闻（如GPT-4.5这种前年的新闻）

## 三、推特Trend

| 地区 | 要求 |
|------|------|
| 🇺🇸 美国 | 当日所有Trend |
| 🇯🇵 日本 | 当日所有Trend |

**❌ 取消**: 欧洲Trend、韩国Trend

### 数据源
- Twitter API 或爬虫
- 需要获取完整Trend列表，不是精选

## 四、新浪/百度热搜榜

- 新浪热搜 或 百度热搜（二选一或都包含）
- 当日热搜榜单

### 数据源
- 新浪热搜API/爬虫
- 百度热搜API/爬虫

## 五、游戏新闻（重点关注）

### 5.1 PC独立游戏
- CK3（十字军之王3）
- RimWorld
- Minecraft
- Mount & Blade（骑马与砍杀）
- 太吾绘卷
- 模拟人生类
- Steam新品

### 5.2 同人JRPG
- DLsite动态
- 同人游戏发布

### 5.3 手游/二游
- 米哈游：原神、崩坏、绝区零
- 鸣潮
- 终末地
- 蓝色星原
- 异环
- 无限大
- 行业动态、新游发布

### 数据源
- Steam新闻API
- 游戏媒体网站爬虫
- 手游数据分析平台

## 六、每日一曲

### 6.1 优先级
1. 日语/Vocaloid
2. EDM
3. 华语流行
4. 小语种

### 6.2 内容要求
- 歌曲名
- 歌手
- 曲风
- 背景介绍/推荐理由

---

## 七、行业英语词汇

- 数量：5个
- 要求：高阶行业词汇（非基础词）
- 包含：英文单词、中文释义、例句

---

## 八、格式要求

### 8.1 整体结构
- **双层折叠结构**:
  - 一级：分类标题（可展开/收起）
  - 二级：新闻标题（可展开/收起）
  - 三级：详细内容

### 8.2 技术规格
- 格式：HTML单文件
- 样式：内嵌CSS
- 适配：手机端优先
- 交互：JavaScript实现折叠效果

### 8.3 视觉要求
- 关键数据高亮显示
- 涨跌用颜色区分（涨-红/跌-绿或按惯例）
- 清晰的视觉层次

## 十、输出要求

3. 推送到GitHub仓库
4. 发送给用户确认
